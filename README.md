## 🛡️ Understanding how it works

DNS acts as the phonebook of the internet. It translates website names into numbers. Some networks block these requests to prevent you from seeing certain sites. DNSly wraps your requests in a tunnel. This tricks the network into thinking your traffic is standard background data. Because the filter ignores this data, you gain access to the restricted content. 

This process runs in the background. You do not need to change your settings once you turn the app on. The software manages the connection path for you.

## 🚀 Common troubleshooting steps

If you cannot connect, try these steps:

1. Check your internet connection. Open a web browser and try to load a page.
2. Restart the DNSly app. Close the window and click the icon again.
3. Check for updates. Visit the download page again to see if a newer version exists.
4. Disable other security tools. Some firewalls block tunnel traffic. Turn them off for one minute to see if the connection improves.
5. Restart your computer. This clears stuck processes that might interfere with the software.

## 💬 Frequently asked questions

Is this app safe?
Yes. DNSly encrypts your requests. This keeps your activity private from your internet service provider.

Does this app cost money?
No. DNSly is a free tool. We believe in open access to information.

Will this work on my school or work network?
Yes. It works on most secured networks. However, some advanced networks detect tunneling. If you encounter issues, the app notifies you in the status bar.

Does DNSly slow down my speed?
You might notice a tiny drop in speed. This is normal because your data travels to a secure DNS server before reaching its destination. Most users do not notice any change in daily tasks like reading or browsing.

What do I do if the app crashes?
Restart the app. If it crashes again, look at your desktop for a file named logs.txt. Send this file to our support team for review. This file contains technical details that help us find the cause of the error.
